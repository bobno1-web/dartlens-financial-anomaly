# DESIGN_SYSTEM.md — DARTLens 디자인 시스템

> 표시 layer의 **통일 기준**. 색·타이포·간격·아이콘·컴포넌트·절대원칙을 한 곳에 모은다.
> 정본은 이 문서이며 `static/css/app.css` 의 CSS 변수와 1:1로 대응한다.
> 판정 색 규칙은 `harness/DESIGN_INVARIANTS.md` INV-8을 따른다(초록/빨강 금지). 값·판정은 표시 layer가
> 바꾸지 않는다(INV-7) — 색·배지·그래프는 이미 계산된 값을 **읽어서** 표현만 한다.

## 0. 절대 원칙 (위반 금지)

- **회색 꽉찬 배경 금지** — 페이지 바탕은 따뜻한 near-white(#fcfbf9), 카드는 흰색.
- **카드 왼쪽 세로선(액센트 바) 금지.**
- **무거운 테두리 금지** — 0.5px 효과의 저대비 선(rgba)만. 구분은 선이 아니라 **여백**으로.
- **초록/빨강 금지** — 좋음·나쁨·오류의 결론색을 쓰지 않는다. 경고=앰버, LOW=파랑, 중립=무채색.
- 표시는 **결론이 아니라 검토 후보** — 색·배지·그래프로 판단을 단정하지 않는다.

## 2. 타이포

- 폰트: **Pretendard** (CDN, 오프라인 시 시스템 폰트로 graceful fallback).
- weight: **400 / 600 / 700 만** 사용(그 외 금지).
- 헤딩 `letter-spacing` -.02 ~ -.03em, 본문 `line-height` 1.55.
- 숫자: `font-variant-numeric: tabular-nums` (자릿수 정렬, 값 비교용).

## 3. 간격 · radius

- radius: **카드 12 · 입력 9 · 버튼 8~9 · 배지 20(=999 pill) · 막대 4 · 로고 7**.
- 카드 padding 1.3~1.5rem, 섹션 상하 여백 ~2.2rem, 히어로는 더 넉넉히.
- 그림자: 저대비 2겹(`--shadow`), 작은 것은 `--shadow-sm`. 강한/짙은 그림자 금지.

## 4. 아이콘

- **Tabler Icons (ti)**, **아웃라인** 스타일(채우기 금지, stroke ~2px).
- 로고 아이콘 = **ti-lens**(돋보기/렌즈). 차콜 배경(`#26241f`) 위 흰색, radius 7.

## 5. 컴포넌트

- **카드**: 흰 배경 · 0.5px 저대비 선 · radius 12 · 저대비 그림자. 왼쪽 세로선 금지.
- **배지(판정)**: pill(999). 신호색 배경+텍스트+얇은 테두리. 경고/주의엔 `⚑`.
- **버튼**: primary = 차콜(`#26241f`) 흰 글씨 · 보조 = 흰 배경 + 선. radius 8~9.
- **요약 stat**: 라벨(작게) → 값(2rem/700) → **설명 한 줄**. 경고>0이면 값만 앰버.
- **위험 항목 카드**: 배경 `#fdf7ec` + 앰버 테두리. 이름+배지 → 우리/산업 막대 → percentile 캡션.
- **미니 막대(우리 vs 산업)**: 두 값을 **같은 척도(둘 중 |max|)로 정규화** — 왜곡 없이 같은 기준.
  음수는 **중앙 0선** 기준 좌/우. 라벨은 **엔진 실값**(지어낸 값 금지). 계산 제한 항목은 막대 없이 사유만.
- **안내 박스**: 배경 `#f6f8f9`, 결과 상단에 **항상 표시**.
- **정상 접기**: `<details>` — 기본 접힘, 펼치면 전부 표시(은폐 아님, INV-5 준수).
- **푸터**: 전 페이지 공통. 상단 0.5px 구분선(`#efece5`), 가운데 11px `#b0aca4`.

## 6. 구현 메모 (현재 ↔ 통일 목표)

- 아이콘: 현재 inline SVG 렌즈(아웃라인) → **Tabler `ti-lens`** 로 통일 예정(형태 동일).
- 카드 radius: 일부 14px(`--radius`) → **12** 로 수렴 예정.
- 입력/버튼 radius: 현재 10px → **9 / 8~9** 로 수렴 예정.
- 이 문서가 정본이다. 새 컴포넌트는 위 값으로 만들고, 기존 값은 점진적으로 수렴시킨다.
